# EduTech Academy

A responsive website for an online learning platform built with HTML, Bootstrap 5, and custom CSS.

## Features
- 4 responsive pages (Home, About, Courses, Contact)
- Bootstrap 5 components
- Custom CSS styling
- Mobile-first design

## Technologies Used
- HTML5
- Bootstrap 5
- CSS3
- Font Awesome Icons

## Setup
Clone the repository and open any HTML file in your browser.
